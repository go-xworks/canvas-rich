安装与本地运行

npm install
npm run dev
npm run build
npm test

仓库包含核心库与示例站两层：src/ 输出可发布库，examples/ 是 Vite 静态站。

基础用法

import { createEditor } from 'canvas-rich';
import 'canvas-rich/style.css';

const editor = createEditor(document.getElementById('app')!, {
  initialMarkdown: '# Hello\n\nStart typing…',
  theme: 'light',
  viewMode: 'web',
});

调用方只需提供容器，编辑器外壳、Canvas、工具栏和输入代理由库内部创建。

实例 API

  • exec(commandId)：执行加粗、插入表格、撤销等命令。
  • getHTML / setHTML、getMarkdown / setMarkdown、getJSON / setJSON：多格式读写。
  • on / off：监听 doc:changed 等事件。
  • setTheme / setZoom / setViewMode：运行时切换界面状态。

运行资产

默认 Canvas shaper 无额外要求；若启用 HarfBuzz，请将 Roboto / Noto 字体部署到站点根路径 /fonts/。公式渲染需由调用方引入 katex/dist/katex.min.css。

GitHub Pages 部署

npm run build:demo
# 将 dist-demo 发布到 gh-pages 分支或 GitHub Pages artifact

当前站点是多页面 Vite 应用：/ 为官网，/docs.html 为文档官网，/demo.html 为完整编辑器 Demo。

架构层级

ui → editor → text → render → model → shared

核心层保持向下依赖：模型、文本排版、渲染后端、编辑命令和 UI chrome 互相解耦，方便按业务扩展。
