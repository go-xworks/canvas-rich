Capabilities

为专业文档编辑而设计

从渲染、排版、输入到导入导出都由内核统一控制，减少 DOM contenteditable 的不可预测差异。

01

Renderer

字形进入多页 atlas，通过 WebGL2/WebGPU 批量绘制，静态帧零额外重排。

02

Layout

支持连续滚动和类 Word A4 分页，缩放、目录和打印导出在同一模型上工作。

03

Typography

内置 BiDi、HarfBuzz 整形接口、字体回退、嵌套列表和表格布局。

04

Document Model

块行为、导出器、工具栏贡献集中注册，适合沉淀业务级文档组件。

05

Editor Shell

跨块选择、拖拽移动、撤销合并、IME 预览、查找替换和触摸手势。

06

Static Delivery

官网、文档和 Demo 均由 Vite 构建为静态资源，可直接部署到 GitHub Pages。
