Capabilities

为专业文档编辑而设计

从渲染、排版、输入到导入导出都由内核统一控制，减少 DOM contenteditable 的不可预测差异。

Renderer

字形、装饰层、选择区进入同一绘制队列，减少 DOM 与 Canvas 混合导致的状态漂移。

  • 多页 glyph atlas
  • 静态帧缓存
  • WebGL2 / WebGPU 边界

Layout

段落、分页、表格和命中测试共享排版结果，支持连续滚动与类 Word 页面视图。

  • A4 pagination
  • table layout
  • zoom aware hit test

Typography

复杂脚本、双向文本、字体回退和公式排版统一进入 shaping 阶段，避免浏览器差异外泄。

  • BiDi
  • HarfBuzz shaping
  • KaTeX formula

Document Model

块、mark、表格和导出器按契约注册，适合沉淀业务级文档组件与编辑策略。

  • typed blocks
  • export adapters
  • toolbar contribution

Editor Shell

输入、选区、拖拽、撤销和触摸手势围绕统一状态机组织，降低编辑行为分叉。

  • cross-block selection
  • IME composition
  • undo merge

Static Delivery

官网、文档和 Demo 均由 Vite 构建为静态资源，发布链路与包消费方式保持一致。

  • Vite build
  • GitHub Pages
  • workspace assets
